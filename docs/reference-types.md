# Reference Types

Different videos need different references.

## Product Video

Ask for:

- Product photo
- Logo
- Packaging
- Color palette
- Any required angle
- Any text that must appear exactly

Consistency lock:

- Same product shape
- Same logo
- Same color
- Same material
- Same scale

## SaaS Video

Ask for:

- App screenshots
- Dashboard state
- Logo
- Brand colors
- User persona
- Feature being shown

Consistency lock:

- Same app UI
- Same dashboard layout
- Same logo
- Same brand color
- Same screen purpose

## Character Video

Ask for:

- Face reference
- Wardrobe reference
- Pose references
- Location reference
- Important props

Consistency lock:

- Same face
- Same wardrobe
- Same hair
- Same body energy
- Same recurring objects

## Fun / Cinematic Video

Ask for:

- Mood
- Location
- Character type
- Genre references
- Any recurring object

Consistency lock:

- Same world
- Same lighting style
- Same main character
- Same story object

## The Reference Rule

If something matters, make it a reference.

If it is only described in text, expect drift.
