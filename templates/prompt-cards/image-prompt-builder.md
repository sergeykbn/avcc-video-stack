# Image Prompt Builder

```text
Use the approved creative brief and references.

Write first-frame and last-frame prompts for each shot.

For each frame include:
- Shot number
- Frame role: first or last
- Subject
- Location
- Camera angle
- Lighting
- Required references
- Continuity locks
- Negative prompt

Keep the wording practical enough to paste into an image model.

End by asking:
Approve these image prompts, or revise?
```
